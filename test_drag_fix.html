Тест: Исправление перетаскивания маркера

Проблема:

При перетаскивании маркера в модальном окне с картой, установленная точка геопозиции не переносилась на маркер.

Решение:

Добавлены обработчики событий для синхронизации координат маркера в режиме реального времени:

  • Обработчик drag для обновления координат в реальном времени при перетаскивании
  • Обновление адреса в реальном времени с помощью getAddressFromCoords
  • Синхронизация маркера с основной картой при перетаскивании
  • Обновление масштаба основной карты для отображения изменений

Внесенные изменения в /workspace/main.html:

Строки 2774-2793: Улучшен обработчик dragend для обновления адреса и синхронизации с основной картой

Строки 2795-2814: Добавлен обработчик drag для обновления координат в реальном времени

Результат:

Теперь при перетаскивании маркера в модальном окне:

  • Координаты обновляются в реальном времени
  • Адрес обновляется в реальном времени
  • Маркер на основной карте синхронизируется с перетаскиваемым маркером
  • Кнопка подтверждения остается активной во время перетаскивания
